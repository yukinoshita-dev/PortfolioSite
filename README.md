# Portfolio Site

福島在住フリーランスエンジニアのポートフォリオサイトです。

## サイトについて

受託開発・業務システム開発・Web制作などのご依頼・ご相談を承っております。  
サイト内のお問い合わせフォームよりお気軽にご連絡ください。

## 使用技術

| カテゴリ | 技術 |
|----------|------|
| フレームワーク | Next.js 15 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v4 |
| アニメーション | Framer Motion |
| ホスティング | GitHub Pages |
| CI/CD | GitHub Actions |
